【三、当前架构与风格约定（非常重要）】

1️⃣ 严格分层（不可混用）
- Controller：只做参数接收 + Result 返回
- Service：业务逻辑
- Mapper：只做数据库操作
- XML：SQL 全写在 XML 中

2️⃣ 统一注入方式
- 全部使用 @Autowired
- 不使用构造器注入（为了降低理解成本）

3️⃣ Result 规范
- Result.ok(T data)
- 不使用 Result.ok()
- Void 返回统一写：Result.ok(null)

4️⃣ MyBatis 使用规范
- ❌ 不使用 MyBatis-Plus
- ❌ 不使用 selectList / insert 等 MP 方法
- ✅ Mapper 接口方法与 XML id 一一对应

5️⃣ VO / DTO 规范
- Controller 不直接暴露 entity
- VO 用于出参
- DTO 用于入参
